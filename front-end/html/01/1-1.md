# HTML 歷史

- 1993 HTML 1.0
- 1995 HTML 2.0
- 1997-01-14 HTML 3.0
- 1997-12-18 HTML 4.0
- 2000-01-26 XHTML 1.0
- 2007 W3C 認可 HTML5 的標準
- 2019-05-28 W3C 與 WHATWG 合作制定單一版本的HTML和DOM規範
